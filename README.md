##  Dataset

The Iris dataset is a classic dataset in the field of machine learning and statistics. It consists of 150 samples, each with four features:

- **Sepal length** (cm)
- **Sepal width** (cm)
- **Petal length** (cm)
- **Petal width** (cm)

Each sample is classified into one of three species:

- **Setosa**
- **Versicolor**
- **Virginica**
